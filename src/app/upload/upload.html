O que é o bucket? O bucket é como uma pasta no MinIO onde seus arquivos serão armazenados. Cada microserviço pode enviar arquivos para um bucket diferente — basta informar o nome desejado. Se o bucket ainda não existir, ele será criado automaticamente.

Este é um projeto de demonstração. Não envie arquivos sensíveis ou confidenciais.
